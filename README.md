# Game Theory Arena

**Play it here: https://lokigod69.github.io/game-theory-arena/**

A local-first arena for game-theory cartridges — no backend, no accounts, everything runs in your browser:

- **Scapegoat Vote** — a social-deduction elimination game of secret pressure, alliances, and betrayal against 13 heuristic AI personas.
- **Sealed Bid Siege** — a sealed-bid territory auction where every bid burns.
- **Commons & Ostracism** — a public-goods pool with peer punishment: feed the pool or feed on it, and point a finger before the table points at you.

This repository contains only the built static site (Next.js `output: "export"` artifacts). The source tree is developed locally and is not published here.
